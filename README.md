# WhatsApp BOT 🚀

Enterprise WhatsApp automation bot built using whatsapp-web.js.

## Features
- Source group gated forwarding
- Target group mapping
- Random delay anti-spam logic
- Config-driven behavior
- Temp files excluded from Git

## Tech Stack
- Node.js
- whatsapp-web.js
- Git version control

## Status
Phase 1 Complete ✅

## Setup

```bash
npm install
node main.js

## Important Notes

- This bot is for internal HR automation.
- WhatsApp session data is excluded via .gitignore.
- Temporary runtime files are not tracked in Git.
- Do not upload auth or cache folders.
